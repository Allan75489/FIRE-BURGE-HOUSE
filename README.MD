# 🔥 Fire Burge House

Sistema web de **cardápio digital para hamburgueria**, desenvolvido para permitir que clientes visualizem produtos, adicionem itens ao carrinho e realizem pedidos de forma simples.

O projeto simula o funcionamento de um **cardápio online moderno**, com interface organizada e funcionalidades básicas de pedido.

---

# 🍔 Funcionalidades

- 📋 Visualização do cardápio com categorias
- 🔥 Página de promoções
- 🛒 Carrinho de pedidos
- 💰 Cálculo automático do total
- 🧭 Botões de navegação entre páginas
- 🎨 Interface moderna com CSS
- ⚙️ Estrutura de painel administrativo (Admin)

---

# 🖥️ Tecnologias Utilizadas

- **HTML5** – Estrutura das páginas  
- **CSS3** – Estilização e layout responsivo  
- **JavaScript** – Lógica do carrinho e interações

# 🛠️ Painel Administrativo

O projeto também possui uma estrutura inicial de **painel administrativo**, responsável por simular o gerenciamento interno do sistema da hamburgueria.

Esse painel representa como funcionaria a área utilizada pelos administradores para acompanhar pedidos e dados do sistema.

### 📂 Páginas do Admin

- **Login** → acesso ao painel administrativo  
- **Dashboard** → visão geral do sistema  
- **Pedidos** → página destinada à visualização de pedidos realizados  
- **Relatório** → área de análise e estatísticas

### 🎯 Objetivo do Admin

O painel administrativo serve como base para futuras funcionalidades como:

- 📦 Gerenciamento de produtos
- 🧾 Controle de pedidos
- 📊 Relatórios de vendas
- 👥 Controle de usuários administradores

Essa estrutura permite que o projeto evolua para um **sistema completo de gerenciamento de pedidos para restaurantes**.

---

# 🛒 Como Funciona

1. O usuário acessa a página **Home**.  
2. Pode navegar até o **Cardápio**.  
3. Escolhe os produtos e adiciona ao **Carrinho**.  
4. O sistema calcula automaticamente o **total do pedido**.
